BRAINFUCK on RUST and PYTHON
============================

RUST READY:
- [ ] Terminal gui
- [ ] I / O ( , / . )
- [ ] + / -
- [ ] Cycles ( '[]' )
- [ ] > / <

PYTHON READY:
- [ ] Terminal gui
- [ ] I / O ( , / . )
- [ ] + / -
- [ ] Cycles ( '[]' )
- [ ] > / <

INSTALL:
```sh
git clone https://github.com/shuqite/brainfuck
```

TO RUN RUST:
```sh
cd brainfuck/brainfuck-rust
cargo run
```

TO RUN PYTHON:
'''sh
cd brainfuck/brainfuck-python
python brainfuck.py
'''
